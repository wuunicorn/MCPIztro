# 更新日志

本文件记录了项目的所有重要变更。

## [1.0.0] - 2024-08-01

### 新增
- 🎉 初始版本发布
- ✨ 支持完整的紫微斗数排盘计算
- ✨ 支持运限分析（大限、流年等）
- ✨ 支持获取当前系统时间
- ✨ 支持阳历和农历生日输入
- ✨ 支持多语言（简体中文、繁体中文、英文、日文、韩文、越南文）
- ✨ 完整的 MCP (Model Context Protocol) 支持
- ✨ 基于 iztro v2.5.3 库的精确计算
- 📝 完整的 API 文档和使用示例
- 🧪 包含测试脚本验证功能

### 功能特性
- **get_current_time**: 获取详细的当前时间信息
- **calculate_ziwei**: 计算完整的紫微斗数星盘
- **get_horoscope**: 获取特定时间的运限信息

### 技术栈
- Node.js ≥18.0.0
- iztro v2.5.3
- MCP Protocol 2024-11-05
